== Description ==

Binde Werbebanner oder Affiliate Links automatisch in deine Beitr&auml;ge ein.
Standartm&auml;&szlig;ig wird dein Banner automatisch nach dem ersten Absatz gesetzt.
Es besteht die M&ouml;glichkeit f&uuml;r Handynutzer (z.b. Iphone) einen anderen Banner anzeigen zu lassen als f&uuml;r Desktop User, somit kannst du auch Einnahmen durch Mobilen User
gewinnen. 
Willst du den Banner manuell in die Beitr&auml;ge einbinden, dann kannst du dein eigenes Trennzeichen w&auml;hlen und dies beim schreiben deines Beitrages verwenden.
Die CSS Gestaltung vor und hinter dem Banner kann komplett ver&auml;ndert werden.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Plugin in das Verzeichnis wpcontent/plugins hochladen. 
2. Im Menü 'Plugins' aktivieren
3. Unter Einstellungen im Bereich 'Advertise in Text' die gewünschten Einstellungen wählen
4. Fertig
